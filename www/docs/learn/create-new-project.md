## Create app

```bash
npx create-sst@latest my-sst-app
```

Install dependencies

```bash
cd my-sst-app
npm i
```

Start local environment

```bash
npm start
```

The initial deploy can take up to 5-10 minutes to complete. While you wait, lets take a look at the project structure.
